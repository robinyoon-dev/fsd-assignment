# fsd-assignment

Feature-Sliced Design 아키텍처를 적용한 날씨 프로젝트입니다.


## 기술 스택
- **React 19**
- **TypeScript**
- **Vite**
- **TanStack Query** 
- **React Router** 
- **Tailwind CSS**

## 프로젝트 구조

이 프로젝트는 React + TypeScript + Vite를 사용하며, Feature-Sliced Design(FSD) 아키텍처를 따릅니다.

```
fsd-assignment/
└── weather-project/    # 메인 프로젝트 디렉토리
```

## 사전 요구사항

- Node.js 18.x 이상
- npm

## 설치 방법

1. 프로젝트 클론 또는 다운로드 후, 프로젝트 디렉토리로 이동합니다:

```bash
cd weather-project
```

2. 의존성을 설치합니다:

```bash
npm install
```

## 실행 방법

### 개발 서버 실행

개발 모드로 프로젝트를 실행합니다:

```bash
npm run dev
```

개발 서버가 실행되면 브라우저에서 `http://localhost:5173` (또는 터미널에 표시된 주소)로 접속할 수 있습니다.

### 프로덕션 빌드

프로덕션 빌드를 생성합니다:

```bash
npm run build
```

빌드된 파일은 `dist` 디렉토리에 생성됩니다.

### 빌드 미리보기

빌드된 프로젝트를 로컬에서 미리볼 수 있습니다:

```bash
npm run preview
```


## 기타 명령어

### 린트 실행

코드 린트를 실행합니다:

```bash
npm run lint
```

## 구현된 기능


### 1. 현재 위치 감지 기능 (Detect Location)

- 브라우저 Geolocation API를 활용하여 앱 최초 진입 시 사용자의 현재 위치를 감지
- 위치 권한 거부, 타임아웃 등 기본적인 에러 케이스에 대한 분기 처리
- TanStack Query를 활용하여 위치 정보 요청 상태(로딩 / 성공 / 실패)를 관리
- 위치 정보가 준비되기 전까지의 상태를 제어하기 위한 게이트 컴포넌트 구조 적용

> 현재 위치 기반 날씨 조회 기능은 전체 구현을 목표로 설계되었으나,
> 시간 제약으로 인해 위치 감지 로직까지만 구현되었습니다.

## 프로젝트 구조 및 설계 방향
Feature-Sliced Design(FSD) 아키텍처 적용

본 프로젝트는 요구사항을 기준으로 Feature-Sliced Design(FSD) 아키텍처를 적용했습니다.

- app: 애플리케이션 초기화 및 전역 설정
- pages: 라우트 단위의 페이지 컴포지션
- features: 사용자 행동 단위 기능 (예: 위치 감지)
- entities: 도메인 단위 모델 (예: 날씨 데이터)
- shared: 공통 UI, 유틸리티, 설정

> 실제 기능 구현은 제한적이지만,
> 과제 요구사항(검색, 즐겨찾기, 날씨 조회 등)이
> 기능 단위로 자연스럽게 확장될 수 있도록 구조를 설계했습니다.




## 기술 선택 이유 - Next.js 대신 React를 선택한 이유
본 과제 이전까지의 프로젝트에서는 Next.js의 장점을 이해하고 이를 활용하는 환경에 익숙하게 작업해 왔습니다.

다만 이번 과제에서는 프레임워크의 기능을 활용하는 것보다, 아키텍처 설계와 구조적인 접근을 더 중요하게 고려했습니다.
이에 따라, **Feature-Sliced Design(FSD) 아키텍처를 명확하게 적용하고 유지하는 것을 기술 선택의 우선순위로 두었습니다.**

Next.js는 라우팅, 데이터 패칭, 서버 컴포넌트 등 다양한 개념이 프레임워크 차원에서 결합되어 있어 FSD의 레이어 경계를 명확하게 드러내는 데 오히려 복잡도가 증가할 수 있다고 판단했습니다.

따라서 이번 과제에서는:
- React + TypeScript 기반의 클라이언트 애플리케이션을 선택하여
- 기능(feature), 도메인(entity), 페이지(page) 단위의 책임을 명확히 분리하고
- FSD 아키텍처의 의도를 코드 구조에서 최대한 드러내는 데 집중했습니다.
